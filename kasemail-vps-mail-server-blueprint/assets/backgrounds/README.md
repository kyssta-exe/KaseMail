# Background Assets

These background PNGs are extracted from the premium mockups as art-direction assets. They are useful for humans and visual designers. DeepSeek should not depend on seeing them.

For code implementation, recreate the ambience with CSS radial gradients, glows, blur layers, and optional subtle texture, using the recipes in:

- `../../deepseek-text-only/01_text_only_visual_spec.md`
- `../../deepseek-text-only/03_component_and_css_recipes.md`
