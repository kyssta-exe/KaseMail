# Prompt - Security Review

Perform a security review of KaseMail. Check RBAC bypasses, open relay risk, SSRF in DNS checks, API key leakage, mailbox impersonation, session fixation, CSRF, XSS in HTML email rendering, attachment handling, audit log integrity, secret storage, backup encryption, and admin break-glass flow. Produce findings with severity, exploit path, fix, and regression test.
