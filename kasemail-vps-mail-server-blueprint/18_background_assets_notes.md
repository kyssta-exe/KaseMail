# 18 - Background Assets Notes

This folder includes extracted visual background assets derived from the updated premium mockups.

## Location

- `assets/backgrounds/`

## Included files

- `01-login-hero-background.png` - left-side login hero reference background.
- `02-dashboard-ambient-background.png` - admin dashboard ambient shell reference.
- `03-webmail-sidebar-ambient-background.png` - sidebar/webmail glow reference.
- `04-mobile-showcase-background.png` - mobile presentation background reference.
- `05-security-panel-ambient-background.png` - security dashboard ambient reference.

## Intended use

Use these as:

- design references in Figma
- texture / ambient inspiration for product UI backgrounds
- source material for AI-assisted design iterations
- visual guidance for front-end implementation of layered gradients, glows, and atmospheric sections

These are not meant to be directly used as final production backgrounds without optimization, but they are useful as art direction assets.
